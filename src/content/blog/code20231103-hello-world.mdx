# Hello World
多种编程语言输出 "Hello world!"
# Program & Compile

- shell

```sh

echo Hello world
```

- python

```python
from icecream import ic

print("Hello world!")
ic("Hello world!")
```

- rust

```bash

cargo new hello
cd hello
vi src/main.rs
```

```rust

fn main() {
    println!("Hello, world!");
}
```
```sh

cargo run
```

- go

```go

package main

import "fmt"

func main() {
    fmt.Println("hello world")
}
```
```shell

go run hello.go
```

- Kotlin
```kotlin

package main

fun main() {
    val halo: String = "Hello world!"
    println(halo)
}
```
