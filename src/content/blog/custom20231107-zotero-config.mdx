## scihub
[Sci-Hub](https://www.scihub.net.cn/sci-hub/) 是巴基斯坦的一位女神创办的良心大大的网站，可以免费下载大部分文献，貌似较新的不行。

1. 打开 Zotero 的首选项，点击高级–> 设置编辑器
2. 搜索 extensions.zotero.findPDFs.resolvers，双击
3. 删除默认的 `[]`，并将以下代码粘贴进去并点击 ok

    ```json

    {
        "name":"Sci-Hub",
        "method":"GET",
        "url":"https://sci-hub.ren/{doi}",
        "mode":"html",
        "selector":"#pdf",
        "attribute":"src",
        "automatic":true
    }
    ```

# 插件

  自动重命名，生成新项目，将 PDF 从 Zotero 库同步到您的（移动）PDF 阅读器（例如 iPad，Android 平板电脑等）并从 PDF 文件中提取注释。

- [Better BibTeX](https://retorque.re/zotero-better-bibtex/index.html)
  文献导出引用格式：`(auth+year+"-"+shorttitle.ideographs.select(1,0)).lower`
- Zotero translators
  Zotero translators是一款Zotero的中文网页抓取插件，用于维护各种中文翻译器。不安装这个插件会出现无法抓取知网等中文文献元数据。

  [Zotero translators github 下载及使用说明主页](https://github.com/l0o0/translators_CN)

  解压压缩包，将该目录中所需的js文件复制到 Zotero 的 translators 目录(首选项 -> 高级 -> 打开数据文件夹)

- Jasminum 茉莉花
  [Jasminum](https://github.com/l0o0/jasminum)，一个简单的 Zotero 中文插件，旨在提升 Zotero 中文用户的使用体验。主要功能有：

  1. 中文期刊附件（PDF/CAJ）的元数据抓取，支持添加 PDF/CAJ 时自动抓取元数据功能
  2. 集成 Zotero 中文社区转换器的下载与更新功能
  3. 为知网学位论文 PDF 添加章节书签（依赖于 PDFtk）
  4. 更新中文文献引用次数，核心期刊信息
  5. 合并 / 拆分姓，名，识别文献条目语言，下载知网附件，统一日期格式等

  <img src="/images/post-images/zotero/zotero-jasminum.png" alt="Jasminum" width="67%" />

  主要是为了抓取中文文献的元数据，右键条目->知网助手->抓取知网元数据
