# 磁轴键盘
看敌鸽发的“闻所未闻的磁轴键盘”，顺便了解一下磁轴和霍尔效应。

磁轴键盘是利用电磁原理，通过霍尔感应器与永磁铁感应来实现触发，从而提供更快速和更精准的触发体验。磁轴包括轴心、永磁铁、霍尔传感器和弹簧。
当按键被按下时，磁铁会靠近霍尔传感器，感应到磁场的变化，导致霍尔电压超过阈值并触发相应的动作。松开时，按键原理磁体，磁性开关再次感应到碰场的变化，输入信号停止。

磁轴优势主要在于：
1. 稳定性
2. 灵敏
3. 轻便
2. 长寿命，没有按键摩擦碰撞
3. 触感反馈，没有按键摩擦碰撞
4. 低噪音，没有按键摩擦碰撞

目前磁轴已经很火了，从醉鹿到akko 到现在 299 的江湾，还能再卷卷。

# 霍尔效应
1879 年，麦克斯韦去世，爱因斯坦出生，爱迪生发明了电灯泡。。。
同年，霍尔效应被埃德温・赫伯特・霍尔（Edwin Herbert Hall）发现。

**霍尔效应（Hall effect）**是指当固体导体放置在一个磁场内，且有电流通过时，导体内的电荷载子受到洛伦兹力而偏向一边，继而产生电压（霍尔电压）的现象。

**霍尔效应传感器**是一个换能器，将变化的磁场转化为输出电压的变化。霍尔传感器首先是实用于测量磁场，此外还可测量产生和影响磁场的物理量，例如被用于接近开关、霍尔、位置测量、转速测量和电流测量设备。
