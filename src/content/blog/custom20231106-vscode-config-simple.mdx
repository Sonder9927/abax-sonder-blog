## apperence

tmeme: Everforest

icon(optional): Material Icom Theme

zen: C-k + >zen

## efficiency

project: project manager

select: expand-region

template: template string converter

run: code runner

## lsp

error: error lens

lsp: ["eslint", "sort json objects", "markdonwlint"]
